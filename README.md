# 📊 Sales & Customer Dashboard | Tableau

## 📌 Project Overview

This project is an interactive **Sales and Customer Dashboard** developed using **Tableau** to analyze sales performance, customer behavior, profitability, and business trends.

The dashboard provides business insights through interactive visualizations, helping users monitor key performance indicators (KPIs), identify top customers, and analyze sales performance over time.

> **Note:** This project was recreated by following the Tableau tutorial from the **Data with Baraa** YouTube channel. The dataset used in this project was also provided by the channel. This project is for learning and portfolio purposes only.

---

## 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor customer growth and purchasing behavior.
- Compare yearly sales performance.
- Identify top-performing customers.
- Analyze sales and profit by product subcategory.
- Track sales and profit trends over time.
- Build an interactive Tableau dashboard.

## 📊 Dashboard Features

### 📈 Sales Dashboard

- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales & Profit by Subcategory
- Weekly Sales Trend Analysis
- Weekly Profit Trend Analysis
- Year-over-Year Comparison
- Interactive Filters and Navigation

### 👥 Customer Dashboard

- Total Customers
- Sales Per Customer
- Total Orders
- Customer Distribution by Number of Orders
- Top 10 Customers by Profit
- Customer Growth Analysis
- Year-over-Year Comparison
- Interactive Navigation
